# Personal Portfolio

This is my personal portfolio website, built to present my background, experience, and selected engineering projects.

My main focus is Computer Vision and AI, supported by experience in machine learning, DevOps, web development, and embedded systems. The website also includes a dedicated case study for my bachelor thesis on 3D multi-camera tracking, graded 9.9/10.

The project is built with vanilla HTML, CSS, and JavaScript. It is fully responsive and includes light and dark themes, accessible navigation, interactive project sections, and a local PDF version of my CV.
